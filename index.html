Alors ici toutes mes icônes proviennet de la police d'écriture "iconmonstr-iconic-font" chargé via le css. Pour utiliser la police d'écriture, il suffit juste qu'un élément ait la classe "im", mais ceci n'est pas suffisant, il faut ajouter la classe correspondant à l'icône, c'est un peu comme une lettre. Au-delà de leur poids très faible (un svg c'est du vectoriel), les svg fonts présentes les avantages suivants :
- Personnalisable comme du texte html, on peut donc changer la couleur ou la taille (et ce sans perte) - Possibilité de mettre des transitions, notamment les couleurs ou la taille

  • classes : im im-plane
    style : font-size: 65px; color: red;
  • classes : im im-plane
    style : font-size: 10px; color: purple;

  • classes : im im-link
